# 小记录 (small-notebook)

一款轻量级 Android 日常记录应用，帮助用户快速记录和管理每日事项。

## 功能特性

- **日历视图** — 以月历形式浏览记录，日期上显示记录数量，点击日期即可查看当日所有记录
- **列表视图** — 按日期分组展示全部记录，支持按年/月筛选和升序/降序排列
- **统计分析** — 按月统计记录数量并以图表展示，支持查看每月内容分布及详情
- **记录管理** — 支持添加、编辑、删除记录，添加时可选择日期时间，支持内容预设快捷输入
- **数据导入/导出** — 支持将记录导出为 JSON 文件，也可从 JSON 文件导入记录
- **密码保护** — 可开启启动密码，保护隐私数据
- **云端同步** — 支持 Supabase 数据库配置，实现数据云端同步与离线自动同步
- **数据预加载** — 启动时后台预加载数据，提升页面切换流畅度

## 技术栈

- **语言**：Kotlin
- **最低 SDK**：Android 7.0 (API 24)
- **存储**：SharedPreferences + Gson
- **网络**：OkHttp（Supabase REST API）
- **UI**：ViewPager2 + SmoothBottomBar + RecyclerView
